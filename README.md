# Journal de l'Aventurier — Cassian Vael & Friends

Petit site statique HTML/CSS prêt à publier sur GitHub Pages.

## Structure

- `index.html` : page d'accueil avec l'index des journées
- `jours/jour-01.html`, `jours/jour-02.html`, etc. : pages par journée
- `assets/style.css` : style commun

## GitHub Pages

1. Créer un repo GitHub.
2. Ajouter tous les fichiers à la racine du repo.
3. Aller dans `Settings > Pages`.
4. Source : `Deploy from a branch`.
5. Branch : `main`, folder `/root`.
6. Ouvrir l'URL GitHub Pages générée.
